+++ title = "О проекте SimplePhysics" date = 2026-04-30 draft = false +++ ## 🎯 Актуальность В современном мире сложно привлечь внимание абитуриентов к фундаментальным наукам. Физика часто ассоциируется со сложными формулами и скучными лекциями. **SimplePhysics** ломает этот стереотип. Проект решает проблему: - 📉 Низкий интерес школьников к физическим специальностям - 🎮 Отсутствие современных образовательных инструментов в игровом формате - 🔗 Разрыв между наукой и реальными увлечениями молодёжи --- ## 🎮 Что такое SimplePhysics? **SimplePhysics** — это визуальная новелла, где игрок погружается в историю, решая физические задачи по ходу сюжета. Вместо скучных уравнений — захватывающие диалоги и выборы, влияющие на развитие событий. ### Жанр проекта: - Визуальная новелла - Образовательная игра - Интерактивная история ### Целевая аудитория: - Школьники 14–18 лет - Абитуриенты физических факультетов - Все, кто хочет полюбить физику --- ## 🧠 Как работает обучение? | Традиционное обучение | SimplePhysics | |:---|:---| | Сухие формулы и задачи | Сюжетные интеграции физики | | Лекционный формат | Интерактивные диалоги | | Индивидуальная работа | История с выборами | | Оценка — тесты | Прогресс через прохождение | > **Главная идея:** Ты не учишь физику — ты проживаешь историю, где физика помогает герою. --- ## 🎯 Основные задачи проекта 1. **Создать прототип визуальной новеллы** с интеграцией физических концепций 2. **Разработать 3 сюжетные ветки** с разными концовками 3. **Внедрить 10+ физических задач** в повествование 4. **Провести тестирование** на целевой аудитории 5. **Подготовить методические материалы** для использования в профориентации --- ## ⚙️ Технологии | Компонент | Технология | |-----------|------------| | Игровой движок | Ren'Py | | Создание артов | Adobe Photoshop / Krita | | Звуковое сопровождение | Audacity / FL Studio | | Сценарий и диалоги | Twine / Ink | | Сайт проекта | Hugo + Markdown | | Версионирование | Git / GitHub | --- ## 📅 План развития ### Этап 1 (Текущий) — Прототипирование -
